# Base Exercício CSS-in-JS — Styled Components

Fork do projeto [ogiansouza/base_exercicio_css_in_js](https://github.com/ogiansouza/base_exercicio_css_in_js) com a estilização convertida para **Styled Components**.

## ✅ Componentes estilizados com Styled Components

| Componente | Styled Components |
|---|---|
| `Header` | `HeaderContainer`, `Logo`, `Nav`, `NavLink` |
| `Hero` | `HeroSection`, `HeroTitle`, `HeroSubtitle` |
| `Formulario` | `FormSection`, `FormContainer`, `FormGroup`, `FormLabel`, `FormInput`, `FormSelect`, `FormButton` |
| `Vaga` | `VagaCard`, `VagaCargo`, `VagaDescricao`, `VagaTag`, `VagaSalario`, `BotaoCandidatar` |
| `ListaVagas` | `ListaSection`, `ListaGrid`, `ListaTitulo`, `ListaContagem`, `SemResultados` |
| `App` | `GlobalStyle` com `createGlobalStyle` |

## ▶️ Como rodar

```bash
npm install
npm start
```
